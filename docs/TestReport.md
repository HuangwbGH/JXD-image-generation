## 建议回归清单

- 未登录访问 `/`、`/records`、`/settings` 会跳转登录。
- 普通用户不能访问设置页和用户管理页。
- 管理员能审核注册用户、禁用用户、启用用户和重置密码。
- 普通用户生成时使用所属部门 API Key。
- 管理员生成时可以选择部门。
- 图生图模式必须上传图片。
- 上传图片数量、大小和 MIME 类型校验生效。
- 生成成功后图片可预览、下载，并写入生成记录。
- 删除记录后数据库记录、上传缓存和输出图片均被删除。
- 系统日志不包含 API Key、密码或图片 base64。

## 风险

- 缺少自动化测试，后续代码变更容易遗漏权限和文件删除回归。
- Gemini 连通性依赖公司代理和部门 API Key，必须在目标网络环境中人工验证。
