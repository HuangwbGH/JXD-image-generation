## 产品定位

JXD Gemini 生图工具面向公司内部用户，统一提供 Gemini 文生图、图生图、部门 API Key 管理、程序级代理、生成记录和系统日志能力。

## 用户角色

- 普通用户：归属于 `研发一部` 或 `研发二部`，只能使用本部门 API Key，只能查看和删除自己的生成记录。
- 管理员：维护用户、审核注册、配置部门 API Key、配置模型与代理、查看全部记录和系统日志。

## MVP 范围

- 本地账号登录和注册审核。
- 文生图与图生图。
- 多图上传，默认最多 8 张，单图默认最大 10MB。
- 模型、比例、分辨率和输出格式选择。
- 部门级 Gemini API Key 加密保存。
- 程序级 HTTP/HTTPS 代理。
- 生成记录、物理删除和系统日志。
- Docker 部署与 Linux 运行。

## 验收摘要

- 未登录用户不能访问主页面、记录页、设置页和受保护文件。
- 普通用户生成时自动使用所属部门 API Key。
- 管理员可以审核注册用户并维护部门 API Key。
- 生成成功后保存记录和图片；删除记录时同步删除相关文件。
- API Key、密码和 Token 不出现在前端响应、日志或文档示例中。
- `npm run lint`、`npm run typecheck`、`npm run build` 可通过。
